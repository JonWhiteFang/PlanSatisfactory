
# Satisfactory Factory Planner

This web application allows players of Satisfactory to plan their factory layouts efficiently by selecting different biomes and tech levels. The application provides a drag-and-drop grid system for designing factory layouts, a production rate calculator, and options for saving/exporting layouts.

## Features:
- **Biome Selection UI**: Choose from different biomes.
- **Machine/Tech Level Filter**: Select machines and technologies available at different stages of the game.
- **Drag-and-Drop Factory Layout Planner**: Plan your factory layout interactively using a grid system.
- **Production Rate Calculator**: Calculate how much material your factory produces/consumes.
- **Save/Export Layout**: Save your factory layout or export it for sharing with others.
- **Game Data Integration**: Pulls data from the **Docs.json** file, ensuring up-to-date recipes and machine information.

## Tech Stack:
- **Frontend**: React, TypeScript, Next.js
- **Styling**: TailwindCSS (or your preferred styling framework)
- **Linting and Formatting**: ESLint, Prettier
- **CI/CD**: GitHub Actions
- **Hosting**: GitHub Pages

## Installation and Setup:
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/satisfactory-factory-planner.git
   cd satisfactory-factory-planner
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app locally:
   ```bash
   npm run dev
   ```
